## Inference Prompts

The inference prompts are stored in [50_data_points_from_alpaca.csv](./50_data_points_from_alpaca.csv), which contains 50 data points sampled from the Alpaca dataset. They are used in the talk to evaluate the performance of Ollama and vLLM on a small benchmark.
